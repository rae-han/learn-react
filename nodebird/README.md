- SSR
화면 전체가 한방에 그려진다는 장점이 있지만 과정이 길기 때문에 로딩 속도가 많이 걸릴수 있다.
요즘은 3초 안에 화면 못그리면 사람들이 떠난다?
프리랜더 서버사이드 랜더링 2가지가 있다.
프리랜더는 검색 엔진은 SSR로 일반 유저는 리액트로 


- SPA
